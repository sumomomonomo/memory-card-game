# 神経衰弱ゲーム

ブラウザで遊べるシンプルな神経衰弱ゲームです。  
カードを2枚ずつめくり、すべて揃えるとクリアになります。

👉 **[ゲームを遊ぶ](https://sumomomonomo.github.io/memory-card-game/index.html)**

---

## 遊び方
- カードをクリックするとめくれます
- 同じ文字のカードを2枚揃えると正解
- すべてのペアを揃えるとクリア
- 「リスタート」ボタンで最初から遊べます

---

## 機能
- カードのシャッフル
- めくった回数のカウント
- クリア時の結果表示
- リスタート機能
- ツイート機能を追加予定です
